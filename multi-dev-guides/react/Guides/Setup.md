Quick setup steps for React single-page applications and common tooling recommendations.

## Environment

- Install Node.js 18+ and npm or yarn.
- Recommended: use `nvm` to manage Node versions.

## Create & run

```
npx create-react-app my-app
cd my-app
npm start
```

## Tooling

- ESLint + Prettier for linting & formatting.
- Storybook for component development.

## Common commands

```
npm test
npm run build
npm run storybook
```

## Troubleshooting

- If dependencies fail, delete `node_modules` and reinstall.
